# 简历制作 · Resume Builder

轻量、免费的在线简历制作工具。选择模板、填写内容、实时预览，一键导出 A4 PDF。

## 功能

- **多模板**：现代风格、经典风格、GitHub 风格（左侧个人信息与链接，右侧经历/技能等）
- **可编辑区块**：基本信息、个人评价、专业技能、工作经历、教育背景、荣誉奖项；支持添加/移除与排序
- **个人链接**：在「基本资料」中添加多个链接（如 GitHub、博客），GitHub 风格模板下在左侧栏展示
- **时间选择器**：教育/工作经历使用月份选择器与「至今」选项
- **实时预览**：左侧编辑、右侧 A4 预览，支持导出 PDF

## 技术栈

- React 19 + Vite 7
- Tailwind CSS v4
- React Router、React Markdown、Lucide React

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:5173`。首页可进入「制作简历」，关于页见「关于作者」。
